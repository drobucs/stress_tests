# stress_tests

## check for equals answers


* `cpl.py <files>`
	* компилирует .cpp файлы переданные в аргументы

* `gen.py`
	* генерирует тесты и тестирует

* `smart.cpp`
	* усное решение которое надо протестировать

* `stupid.cpp`
	* работающее решение

* `logs.txt`
	* логи

* `out_smart.txt`, `out_stupid.txt`
	* понятно что

* `faild_tests.txt`
	* логи *WA* тестов

* `test.txt`
	* сюда генерируются тесты
